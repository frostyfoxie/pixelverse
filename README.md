# 🌌 The Cosmos — Cinematic Journey

An interactive, cinematic storytelling experience that takes you from the birth of the universe to the emergence of human consciousness.

## ✨ Overview

This project is a fully immersive, single-file web experience built using HTML, CSS, and JavaScript.

It visualizes:
- The Big Bang
- Formation of stars and galaxies
- Birth of Earth
- Evolution of life
- Rise of humans

All presented through custom pixel-based animations and transitions.

## 🎯 Features

- Fullscreen cinematic slideshow
- Custom particle systems
- Procedural animations (no libraries)
- Scene-based storytelling
- Interactive transitions

## 🛠️ Tech Stack

- HTML5 Canvas
- Vanilla JavaScript
- CSS3 Animations
